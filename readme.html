Расширения

Формат указания адреса станции - без схемы /u/, но со слешем в конце.
Пример: http://your-ii-station.ru/ii-point.php?q=/

Схема /x/c - это схема для получения количества сообщений эхоконференции, помогает снижать потребление трафика, сохраняет временный файл для каждой станции. Почитать про него можно на странице Расширения в документации (ссылка выше).

Формат прокси распознаётся http и socks. Если стоит socks, то указывается, как хост:порт, если http, то юзер:пароль@хост:порт (пароль и юзер необязательны).

Клиент распространяется под лицензией CC0 Universal (общественное достояние), прочитать её текст можно в файле LICENSE в составе программы или здесь.

Библиотека socks_proxy, лежащая в каталоге socks файлов клиента, не является его частью, и копирайты там указаны отдельно (в исходном коде).
